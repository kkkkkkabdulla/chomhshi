## T16-1 联调验收清单（按顺序执行）

1. 健康检查
   * `GET /api/ping` 返回 `code=200`
2. 用户登录链路
   * `POST /api/user/wxLogin` 成功，拿到 token
   * `GET /api/user/verifyToken`、`GET /api/user/info` 正常
3. 内容发布链路
   * `POST /api/post/publish` 正常发布（状态应为待审核）
   * 敏感词发布应被拦截（返回参数错误提示）
4. 管理审核链路
   * `POST /api/admin/login` 成功
   * `GET /api/admin/post/list?status=0` 能看到待审核
   * `POST /api/admin/post/approve/{id}` 后，帖子在用户端列表可见
5. 互动链路
   * `POST /api/post/like/{id}` 点赞/取消点赞可切换
   * `POST /api/comment/add`、`GET /api/comment/list/{postId}`、`DELETE /api/comment/delete/{id}` 正常
6. 举报链路
   * `POST /api/report/add` 能累计举报次数
   * 达到阈值后帖子自动下架（`status=3`）
7. 后台处理举报链路
   * `GET /api/admin/post/reportList` 可查看举报
   * `POST /api/admin/post/handleReport/{id}` 可处理
   * `POST /api/admin/post/restore/{id}` 可恢复帖子

## T16-2 边界与异常测试清单

* 不带 token 访问需要鉴权接口，返回未登录/过期
* 非本人编辑/删除帖子，返回无权限
* 删除不存在资源，返回资源不存在
* 上传非法文件类型或超大小文件，返回明确错误
* 分页参数异常（page<=0、pageSize过大）时可兜底处理

## T16-5 常见问题排查模板

1. 404：

   * 确认访问 URL 前缀与接口一致
